# Mini Task Manager - Python Project

This is a simple command-line task manager built with Python.  
The project demonstrates basic file handling, modular programming, and version control using Git.

## Features
- Add new tasks
- View all tasks
- Delete tasks
- Count total tasks

## Project Structure

mini-task-manager/
│
├── main.py
├── task_manager.py
├── utils.py
└── README.md

## How to Run

Make sure you have Python 3 installed.

Run the program using:

python main.py

## Technologies Used
- Python 3
- JSON
- Git & GitHub

## Purpose of the Project

The purpose of this project is to practice:
- Creating and managing a GitHub repository
- Cloning a repository locally
- Making multiple commits
- Tracking file changes
- Working with modular Python code
